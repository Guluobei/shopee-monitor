# 产品线定义

本文档定义Shopee竞品监控套件的产品线分类方法论。

## 产品线分类原则

产品线分类基于以下维度：

1. **功能相似性**：同类功能的产品归为同一产品线
2. **价格区间**：不同价位段可能需要分开监控
3. **竞争格局**：竞争对手群体相似的产品归为一类
4. **搜索关键词**：用户搜索行为相似的产品

## 自定义产品线

用户可以在 `config/competitors.yaml` 中添加自定义产品线：

```yaml
product_lines:
  NEW_PRODUCT:
    name: "新产品线名称"
    keywords:
      - "关键词1"
      - "关键词2"
    min_price: 100  # 可选：最低价格筛选
    max_price: 500  # 可选：最高价格筛选
```

## 关键词选择建议

1. **品牌词**：主要竞品品牌名
2. **品类词**：产品类别名称
3. **型号词**：具体产品型号
4. **功能词**：核心功能描述

**注意事项**：
- 关键词不要太宽泛（如"手机"）
- 关键词不要太窄（如具体型号可能搜不到竞品）
- 建议组合使用品牌词+品类词
