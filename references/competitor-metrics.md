# 竞品分析指标定义

本文档定义竞品数据分析中使用的各项指标及其计算方式。

## 核心指标

### 销量指标

| 指标 | 定义 | 计算方式 | 用途 |
|------|------|----------|------|
| 总销量 | 商品累计销量 | 直接采集 | 判断商品热度 |
| 月销量 | 近30天销量 | 直接采集（如有） | 判断近期趋势 |
| 销量增速 | 销量增长率 | (本月-上月)/上月 | 判断增长趋势 |
| 销量占比 | 单品在产品线的占比 | 单品销量/产品线总销量 | 判断单品竞争力 |

### 价格指标

| 指标 | 定义 | 计算方式 | 用途 |
|------|------|----------|------|
| 当前价格 | 商品售价 | 直接采集 | 定价参考 |
| 原价 | 商品原价 | 直接采集（如有） | 折扣分析 |
| 折扣率 | 折扣幅度 | (原价-现价)/原价 | 促销力度 |
| 产品线均价 | 产品线平均价格 | Σ价格/商品数 | 定价基准 |
| 价格偏离度 | 价格与均价差距 | (价格-均价)/均价 | 判断定价策略 |
| 价格区间 | 价格分布范围 | min-max | 市场定位分析 |

### 评价指标

| 指标 | 定义 | 计算方式 | 用途 |
|------|------|----------|------|
| 评分 | 商品评分 | 直接采集 | 质量判断 |
| 评价数 | 评论数量 | 直接采集 | 热度判断 |
| 评分分布 | 评分区间分布 | 统计分析 | 质量分布 |

### 店铺指标

| 指标 | 定义 | 计算方式 | 用途 |
|------|------|----------|------|
| 店铺评分 | 店铺整体评分 | 直接采集 | 店铺质量 |
| 粉丝数 | 店铺关注人数 | 直接采集（如有） | 品牌影响力 |
| 商品数 | 店铺商品数量 | 直接采集（如有） | 店铺规模 |

## 市场份额指标

### 站点市场份额

```
站点市场份额 = 站点销量 / 总销量 × 100%
```

用于判断各站点在整体市场中的重要性。

### 产品线市场份额

```
产品线市场份额 = 产品线销量 / 站点总销量 × 100%
```

用于判断各产品线在站点内的竞争格局。

### 单品市场份额

```
单品市场份额 = 单品销量 / 产品线总销量 × 100%
```

用于判断单品在产品线内的竞争力。

## 竞争强度指标

### HHI指数（赫芬达尔-赫希曼指数）

```
HHI = Σ(市场份额²) × 10000
```

- HHI < 1500：低集中度，竞争激烈
- HHI 1500-2500：中等集中度
- HHI > 2500：高集中度，寡头市场

### CRn指数（集中度比率）

```
CRn = Top n 商品市场份额之和
```

常用 CR3、CR5、CR10，判断头部集中程度。

## 价格竞争力分析

### 价格分位数

| 分位数 | 说明 |
|--------|------|
| P10 | 低价区间（10%商品低于此价格） |
| P25 | 较低价区间 |
| P50（中位数） | 中等价格 |
| P75 | 较高价区间 |
| P90 | 高价区间 |

### 价格竞争力评分

```
价格竞争力 = (P50 - 当前价格) / P50 × 100
```

- 正值：价格低于中位数，有竞争力
- 负值：价格高于中位数，溢价定位

## 热度指标

### 热度评分（综合指标）

```
热度评分 = 销量权重 × 销量得分 + 评分权重 × 评分得分 + 评价权重 × 评价得分
```

建议权重：
- 销量权重：0.5
- 评分权重：0.3
- 评价权重：0.2

### 趋势指标

```
趋势得分 = (本月销量 - 上月销量) / 上月销量 × 100
```

- 正值：上升趋势
- 负值：下降趋势

## 数据清洗规则

### 数值清洗

1. **货币符号**：移除 RM、Rp、฿、₱、S$、₫ 等货币符号
2. **单位转换**：
   - "万" → ×10000
   - "K" → ×1000
3. **千分位**：移除逗号分隔符

### 异常值处理

1. **价格异常**：
   - 价格 < 0：标记为无效
   - 价格 > 产品线均价 × 5：标记为异常高价
2. **销量异常**：
   - 销量 < 0：标记为无效
   - 销量 > 产品线最大销量 × 10：标记为异常

### 去重规则

默认按 `商品名称 + 店铺名称` 去重，保留最新记录。

| 产品线 | string | 产品线中文名 |
| 产品线代码 | string | OP/OM/MIC等 |

| 商品名称 | string | 商品标题 |
| 店铺名称 | string | 店铺名 |
| 价格 | float | 当前售价（本地货币） |
| 原价 | float | 原价（如有） |
| 销量 | int | 累计销量 |
| 月销量 | int | 月销量（如有） |
| 评价数 | int | 评论数量 |
| 评分 | float | 商品评分（0-5） |
| 商品ID | string | Shopee商品ID |
| 商品链接 | string | 商品详情页URL |
| 销量占比(%) | float | 在产品线的销量占比 |
| 价格偏离度(%) | float | 与均价的偏离程度 |
