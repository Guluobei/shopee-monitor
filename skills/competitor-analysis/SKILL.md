# 竞品数据分析

> 分析采集的竞品数据，计算市场指标，生成洞察报告

## 功能

- **数据清洗**：标准化列名、清洗数值、去重
- **市场指标**：市场份额、价格偏离度、销量占比
- **站点统计**：按站点汇总商品数、销量、均价
- **产品线统计**：按产品线汇总竞争格局
- **Top榜单**：热销商品Top10

## 使用方法

### Agent调用

```
分析已有的竞品数据，计算市场份额
```

```
分析data/downloads目录下的Excel文件
```

### 命令行

```powershell
# 处理数据
python scripts/data_processor.py --download-dir ./data/downloads

# 指定输出目录
python scripts/data_processor.py --output-dir ./output
```

## 输入

- **Excel文件**：知虾导出的 `.xlsx` 文件
- **位置**：默认 `data/downloads/` 目录

## 输出

| 输出 | 格式 | 说明 |
|------|------|------|
| CSV报告 | `.csv` | 标准化的完整数据 |
| 分析摘要 | `.json` | 按站点/产品线统计 |

## 分析指标

| 指标 | 计算方式 | 用途 |
|------|----------|------|
| 销量占比(%) | 单品销量/站点产品线总销量 | 判断单品竞争力 |
| 价格偏离度(%) | (价格-均价)/均价 | 判断定价策略 |
| 站点总销量 | 站点所有商品销量之和 | 判断市场规模 |
| 产品线均价 | 产品线所有商品平均价格 | 定价基准参考 |

## 输出字段（CSV）

```
采集日期, 采集站点, 站点代码, 产品线, 产品线代码, 搜索关键词,
商品名称, 店铺名称, 价格, 原价, 销量, 月销量, 评价数, 评分,
商品ID, 商品链接, 销量占比(%), 价格偏离度(%)
```

## 安全边界

- **只读取文件**，不修改原始数据
- **输出到独立目录**，不影响源文件
